Title: Two Kings
Intended Game Platform: Desktop
Game Genre: Fighter
Game Objective: Have more health than the other player to win
Rules: Attack the other player before timer runs out, else tie
Game Mechanics: 
	- Player can attack
	- Player can block the other's attack by countering their attack
	- If attack hits other player, then that player's health is decreased
	- Player can jump
	- Player can move left and right
	- If time runs out, whoever has the most health wins, else tie.
